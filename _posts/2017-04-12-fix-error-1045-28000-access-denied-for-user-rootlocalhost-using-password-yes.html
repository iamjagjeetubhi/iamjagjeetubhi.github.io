Have you set-up a password?

This error pops-up when the password has not been set-up.

You should change your root password after installation if you have not by using following command:

sudo dpkg-reconfigure mysql-server-5.5(Or your version)

Than Enter the new password.

This will change your mysql server password and your error will be gone.
