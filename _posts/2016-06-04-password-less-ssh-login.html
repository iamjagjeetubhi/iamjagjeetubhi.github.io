  • First of all execute the following command:
$ ssh-keygen

The above command is used forÂ authentication key generation.

  • Then you'll need to copy the new key to your server using:
$ ssh-copy-id user@host
  • After the key is copied, SSH into the machine as normal:
$ ssh user@host
  • You can now login without entering a password from the particular machine you executed the commands at. :)
